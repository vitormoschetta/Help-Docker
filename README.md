# Help-Docker

#### Informações sobre o docker instalado:
```
docker info
docker version
```

#### Consultar imagens existentes:
```
docker ps -a
```

#### Consultar imagens existentes no host:
```
docker images
```

#### Consultar imagens existentes no host:
```
docker images
```

#### Baixar uma nova imagem:
```
docker pull <imagem>
```

#### Executar uma imagem:
```
docker run <imagem>
```
> O comando acima apenas executa a imagem e 'morre'. Seria como um _build_.
